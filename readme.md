# EcoRide

Plateforme de covoiturage écologique — projet ECF.

**Documentation complète :** [docs/README.md](docs/README.md)

## Démarrage rapide

- **Docker :** `docker-compose up -d` puis http://localhost:8001
- **WAMP :** placer le projet dans `www`, accéder à `http://localhost/ecoride_ecf`

## Structure

Le code web est dans `public/`. Les scripts SQL sont dans `database/`. La documentation est dans `docs/`. Voir `docs/ARCHITECTURE.md` pour les couches POO.
